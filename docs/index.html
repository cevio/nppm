NPPM

Node Private Package Manager

快速上手 →

简易部署

一键安装、一键升级，可视化安装本服务，提供丰富的后台配置，即改即生效。

轻量服务

专注于提供私有包服务不参杂其他NPM实时同步功能，保障系统稳定与轻量。

兼容性强

兼容NPM默认登录方式和第三方登录方式，同时兼容NPM不同版本的API。

插件化

提供强大的插件嵌入功能和服务事件功能，供开发者自定义企业级应用功能。

可视化

提供强大的可视化管理，增强用户体验，管理员轻松管理系统。

统计化

日榜、周榜、月榜，提供实时数据统计，便于了解模块热度。

直接启动

$ npm i -g @nppm/npm # 安装命令
$ cd <dictionary> # 需要确定一个目录来存放资源
$ nppm --port=3000 # 启动服务

进程守护

$ npm i -g @nppm/npm pm2 # 安装命令
$ cd <dictionary> # 需要确定一个目录来存放资源
$ pm2 start nppm -- --port=3000 # 启动服务
